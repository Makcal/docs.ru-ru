## <a name="example"></a><span data-ttu-id="c7b72-111">Пример</span><span class="sxs-lookup"><span data-stu-id="c7b72-111">Example</span></span>

```csharp

using System.Collections;  
static class Test  
{  
    public static void Main()  
    {
```  
