<span data-ttu-id="c3d6a-104">Метод " \<methodname1> " не может реализовать разделяемый метод " \<methodname2> ", так как " \<methodname3> " уже реализует его.</span><span class="sxs-lookup"><span data-stu-id="c3d6a-104">Method '\<methodname1>' cannot implement partial method '\<methodname2>' because '\<methodname3>' already implements it.</span></span> <span data-ttu-id="c3d6a-105">Только один метод может реализовывать разделяемый метод.</span><span class="sxs-lookup"><span data-stu-id="c3d6a-105">Only one method can implement a partial method.</span></span>  
  
 <span data-ttu-id="c3d6a-106">Нельзя иметь два разделяемых метода, реализующих одно и то же объявление разделяемого метода.</span><span class="sxs-lookup"><span data-stu-id="c3d6a-106">You cannot have two partial methods that implement the same partial method declaration.</span></span> <span data-ttu-id="c3d6a-107">Следующий код вызывает эту ошибку.</span><span class="sxs-lookup"><span data-stu-id="c3d6a-107">The following code causes this error.</span></span>  
  
```vb  
Partial Class Product  
  
    ' Declaration of the partial method.  
    Partial Private Sub ValueChanged()  
    End Sub  
  
End Class  
```  
  
```vb  
Partial Class Product  
  
    ' First implementation of the partial method.  
    Private Sub ValueChanged()  
        MsgBox(Value was changed to " & Me.Quantity)  
    End Sub  
  
    ' Second implementation of the partial method causes this error.  
    'Private Sub ValueChanged()  
    '    Console.WriteLine("Quantity was changed to " & Me.Quantity)  
    'End Sub  
  
End Class  
```
