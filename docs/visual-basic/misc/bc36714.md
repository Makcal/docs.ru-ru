# <a name="expanded-properties-cannot-be-initialized"></a><span data-ttu-id="d9ca4-103">Невозможно инициировать расширенные свойства</span><span class="sxs-lookup"><span data-stu-id="d9ca4-103">Expanded properties cannot be initialized</span></span>

<span data-ttu-id="d9ca4-104">Автоматически реализуемое свойство можно инициализировать в рамках его объявления, а расширенное свойство нельзя.</span><span class="sxs-lookup"><span data-stu-id="d9ca4-104">An auto-implemented property can be initialized as part of its declaration, but an expanded property cannot be.</span></span>  
  
 <span data-ttu-id="d9ca4-105">**Идентификатор ошибки:** BC36714</span><span class="sxs-lookup"><span data-stu-id="d9ca4-105">**Error ID:** BC36714</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="d9ca4-106">Исправление ошибки</span><span class="sxs-lookup"><span data-stu-id="d9ca4-106">To correct this error</span></span>  
  
- <span data-ttu-id="d9ca4-107">Используйте автоматически реализуемое свойство или удалите инициализацию из объявления свойства.</span><span class="sxs-lookup"><span data-stu-id="d9ca4-107">Either use an auto-implemented property or remove the initialization from the property declaration.</span></span>  
  
## <a name="example"></a><span data-ttu-id="d9ca4-108">Пример</span><span class="sxs-lookup"><span data-stu-id="d9ca4-108">Example</span></span>  

 <span data-ttu-id="d9ca4-109">В следующих примерах показаны автоматически реализуемое свойство и расширенное свойство.</span><span class="sxs-lookup"><span data-stu-id="d9ca4-109">The following examples show both auto-implemented and expanded properties.</span></span> <span data-ttu-id="d9ca4-110">Автоматически реализуемые свойства можно инициализировать с помощью присваивания или предложения `New` , а расширенные свойства нельзя.</span><span class="sxs-lookup"><span data-stu-id="d9ca4-110">Auto-implemented properties can be initialized by using assignment or a `New` clause, but expanded properties cannot be.</span></span>  
  
```vb  
Class AutoImplementedExample  
    ' An auto-implemented property can be assigned an initial value.  
    Property IDNum As Integer = 33542  
    ' An auto-implemented property can be initialized with New.  
    Property Name As New String("Don Hall")  
End Class  
  
Class ExpandedExample  
    ' Attempting to expand an initialized auto-implemented property  
    ' causes this error.  
    'Property IDNum As Integer = 33542  
    '    Get  
    '    End Get  
    '    Set(ByVal value As Integer)  
    '    End Set  
    'End Property  
  
    ' Instead, you can assign the initial value to the backing field.  
    Private _IDNum As Integer = 33542  
    Property IDNum As Integer  
        Get  
        End Get  
        Set(ByVal value As Integer)  
        End Set  
    End Property  
End Class  
```
