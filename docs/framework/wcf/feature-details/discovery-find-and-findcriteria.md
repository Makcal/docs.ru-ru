## <a name="findresponse"></a><span data-ttu-id="76d18-141">FindResponse</span><span class="sxs-lookup"><span data-stu-id="76d18-141">FindResponse</span></span>

<span data-ttu-id="76d18-142"><xref:System.ServiceModel.Discovery.FindResponse> имеет свойство коллекции <xref:System.ServiceModel.Discovery.FindResponse.Endpoints%2A>, в котором содержатся все ответы, отправленные соответствующими службами в сети.</span><span class="sxs-lookup"><span data-stu-id="76d18-142"><xref:System.ServiceModel.Discovery.FindResponse> has an <xref:System.ServiceModel.Discovery.FindResponse.Endpoints%2A> collection property that contains any replies sent by matching services on the network.</span></span> <span data-ttu-id="76d18-143">Если ни одна служба не отправила ответ, коллекция пустая.</span><span class="sxs-lookup"><span data-stu-id="76d18-143">If no services replied, the collection is empty.</span></span> <span data-ttu-id="76d18-144">Если ответила одна или несколько служб, все ответы сохраняются в объекте <xref:System.ServiceModel.Discovery.EndpointDiscoveryMetadata>, в котором содержится адрес, контракт и дополнительные сведения о службе.</span><span class="sxs-lookup"><span data-stu-id="76d18-144">If one or more services replied, each reply is stored in an <xref:System.ServiceModel.Discovery.EndpointDiscoveryMetadata> object, which contains the address, contract, and some additional information about the service.</span></span>

<span data-ttu-id="76d18-145">В следующем примере описывается выполнение операции поиска в коде.</span><span class="sxs-lookup"><span data-stu-id="76d18-145">The following example shows how to perform a find operation in code.</span></span>

```csharp
// Create DiscoveryClient
DiscoveryClient discoveryClient = new DiscoveryClient(new UdpDiscoveryEndpoint());

// Create FindCriteria
FindCriteria findCriteria = new FindCriteria(typeof(IPrinterService));
findCriteria.Scopes.Add(new Uri("http://www.contoso.com/building1/floor1"));
findCriteria.Duration = TimeSpan.FromSeconds(10);

// Find ICalculatorService endpoints
FindResponse findResponse = discoveryClient.Find(findCriteria);

Console.WriteLine("Found {0} ICalculatorService endpoint(s).", findResponse.Endpoints.Count)
```
